# Terra View Pro

Sistema inteligente de monitoramento meteorológico desenvolvido para o Projeto Integrador.

---

## Sobre o Projeto

O **Terra View Pro** é uma estação meteorológica inteligente capaz de coletar, armazenar, analisar e exibir dados ambientais em tempo real.

O sistema foi desenvolvido com foco em monitoramento climático, análise de dados e visualização moderna através de dashboards interativos.

A proposta do projeto é unir sensores físicos, processamento de dados e desenvolvimento web para criar uma solução tecnológica aplicada ao monitoramento ambiental.

---

## Funcionalidades

- Monitoramento de temperatura ambiente
- Monitoramento da umidade do ar
- Monitoramento da umidade do solo
- Medição da pressão atmosférica
- Medição da intensidade da luminosidade solar
- Medição do volume de chuva em milímetros
- Histórico climático
- Gráficos em tempo real
- Dashboard interativo
- Sistema de armazenamento de dados
- Envio de dados para servidor
- Previsões climáticas com base em dados anteriores
- Consulta manual de registros antigos

---

## Dashboard

O sistema conta com um dashboard moderno e responsivo contendo:

- Cards meteorológicos
- Gráficos interativos
- Status dos sensores
- Indicadores climáticos
- Histórico de dados
- Página de previsões
- Configurações do sistema
